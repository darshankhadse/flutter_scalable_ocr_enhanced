## 1.0.0

* Initial release
* Added `EnhancedOCRController` for programmatic camera control
* Added `detectOnce` parameter to auto-stop after text detection
* Added `minTextLength` for detection validation
* Added `onTextDetected` and `onDetectionComplete` callbacks
* Added custom `stoppedWidget` support
* Full compatibility with all `flutter_scalable_ocr` features
